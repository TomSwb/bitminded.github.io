# Account Page Features & Implementation Checklist

This checklist tracks the planned features for the BitMinded account page. Tick each box as you complete the implementation.

## Features

- [ ] **Profile Information**

  - [ ] Display username
  - [ ] Display email
  - [ ] Edit profile details (username, email, avatar)

- [ ] **Password & Security**

  - [ ] Change password
  - [ ] Two-factor authentication setup/status
  - [ ] Recent login activity

- [ ] **Subscription & Billing**

  - [ ] Show current subscription plan/status
  - [ ] Payment history
  - [ ] Manage payment methods
  - [ ] Upgrade/downgrade/cancel subscription

- [ ] **App Entitlements**

  - [ ] List unlocked/locked apps
  - [ ] Quick links to each app

- [ ] **Notifications & Preferences**

  - [ ] Email notification settings
  - [ ] Language preference
  - [ ] Theme preference

- [ ] **Account Actions**
  - [ ] Sign out button
  - [ ] Delete account (with confirmation)
  - [ ] Export account data
