# Utility Tools

Helpful tools and test files for working with Supabase.

## What's Here

- `find-all-users.sql` - Query to list all users with details
- `supabase-test.html` - HTML tool to test Supabase connection

## Usage

### Find All Users
```sql
-- Copy from find-all-users.sql and run in SQL Editor
-- Works in both dev and prod
```

### Test Supabase Connection
1. Open `supabase-test.html` in browser
2. It will show which environment you're connected to
3. Useful for verifying environment detection

## Safe for Both Environments

These tools are read-only queries and test pages.
Safe to use in both dev and production.
